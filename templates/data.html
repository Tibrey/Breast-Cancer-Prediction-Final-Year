The system uses the dataset developed by the Wisconsin Hospital for breast cancer research which is a significant and valuable approach to understanding this present global issue in the field of health sector.
  1. Dataset Source and Collection
    • Through an interactive image processing, the Wisconsin Hospital has developed this dataset that contains the information about the patient with cancerous tissues and non-cancerous tissues.
    • The healthcare professional use the Fine Needle Aspirate (FNA) to collect sample from suspicious areas identified through the CT Scan or UltraSound. FNA involves inserting a thin, hollow needle into the suspicious tissue to extract cells for examination.
  2. Pathological Analysis
    • The obtained samples are sent for further analysis to pathologist for detailed analysis under a microscope. They examine the cellular structures to identify the abnormalities that may indicate the cancerous tissues.
    • The goal of the analysis is to detect characteristics such as shape , size and arrangement of cells nuclei which provide the valuable insights of potential presence of malignancy.
  3. Feature Computation
    • The features are computed for each nucleus based on the microscopic analysis. These features includes the textural characteristics of cell nuclei.
    • Example of computed features may include nucleus size, shape,texture and other measurement that helps to characterize the cell abnormalities.
  4. System Implementation
    • Our system make use of those feature to develope a model for breast cancer prediction.
    • Machine Learning techniques are employed to train the system on the dataset enabling it to identify patterns and correlations between the computed features and the presence or absence of breast cancer.
  5. Significance of the System
    • The developed system has the potential to assist for the early detection of breast cancer by providing an accurate diagnosis with high accuracy.
    • Early detection is crucial for successful treatment and improve patient outcomes. The automated system can aid in the quick and accurate analysis of large datasets, providing valuable support to healthcare professionals who need to monitor multiple patients at once.

Model Implemented

This system is developed using the logistic regression. Logistic regression is widely used model in machine learning for binary classification. It models the probability that a given input belongs to a particular class. The logistic regression model function is represented as:

f( w, b( x ) ) = g( w · x + b )

In this equation, fw,b(x) represents the predicted probability,w is the weight vector,bis the bias term, x is the input feature vector, and g (z) is the sigmoid function:

g( z ) = 1/1 + e-z

Objective:

Goal: Find the optimal values for the parameters (w, b) that provide accurate class probabilities.

Forward Pass:

Computation: Compute the linear combination of input features x with the weight vector w and the bias term b.

Activation: Apply the sigmoid function to the result.

Equation: Z = x ⋅ w + b

A = σ(Z)

Cost Function:

Measurement: Assess the error between predicted probabilities and true labels using binary cross-entropy loss.

Equation:

J(w, b) = -1/m ∑[yi log(fw,b(xi)) + (1-yi) log(1-fw,b(xi))] 
J(w, b) = -1/m ∑[yi log(fw,b(xi))
+ (1-yi) log(1-fw,b(xi))]

where m is the number of samples, yi is the true label, and fw,b(xi) is the predicted probability.

Backward Pass (Gradient Computation):

Compute Gradients: Calculate the gradients of the cost function with respect to parameters (w, b).

Equations:

∂J/∂b = 1/m ∑(fw,b(xi) - yi)
∂J/∂w = 1/m ∑(fw,b(xi) - yi)xi

Training Process:

Iterative Optimization: Update parameters (w, b) using an optimization algorithm (e.g., gradient descent).

Update Equations:

w ← w - α ∂J/∂w
b ← b - α ∂J/∂b

where α is the learning rate.

Iteration:

Repeat: Iterate through the training process, performing forward and backward passes, to minimize the cost function.
